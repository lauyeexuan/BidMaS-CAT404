BidMaS AI Project Recommendation API

This service provides AI-powered project recommendations for students based on their interests.

API Endpoints

POST /api/recommend

Get project recommendations based on student interests.

Request Format:

{
    "interests": "JavaScript full-stack development",
    "projects": [
        {
            "id": "1",
            "Title": "Project Title",
            "Description": "Project description",
            "Technologies": ["JavaScript", "React", "Node.js"]
        }
    ]
}

Response Format:

{
    "recommendations": [
        {
            "id": "1",
            "Title": "Project Title",
            "Description": "Project description",
            "Technologies": ["JavaScript", "React", "Node.js"],
            "matchScore": 0.85
        }
    ],
    "processingTime": 234,
    "count": 1
}

For more information, visit the GitHub repository.
